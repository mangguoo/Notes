1. 三角函数：

   ``` js
   弧度定义：弧弧长等于半径的弧，其所对的圆心角为1弧度。
   
   弧度 = Math.PI/180*角度
   Math.sin(弧度)
   ```

2. 平方根、立方根：

   ``` js
   Math.sqrt(9)
   Math.pow(8, 1/3)
   ```

3. 三目运算符要比abs效率高：

   ``` js
   let num = -2
   Math.abs(num)
   num = num < 0 ? -num : num
   ```
